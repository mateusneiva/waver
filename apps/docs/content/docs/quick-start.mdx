This guide will get you from zero to playing music in your Discord server in just a few steps.

## Prerequisites

Before you begin, make sure you have:

- A Discord account.
- A Discord server where you have **Manage Server** permissions (to invite bots).
- A voice channel to play music in.

## Step 1: Invite Waver

Click the invite link on [waverbot.com](https://waverbot.com) to add Waver to your server.

When prompted, select your server and grant the following permissions:

| Permission | Why it's needed |
|---|---|
| **View Channels** | So Waver can read your commands in text channels. |
| **Send Messages** | So Waver can reply with now-playing info, queues, and errors. |
| **Connect** | So Waver can join voice channels. |
| **Speak** | So Waver can play audio in voice channels. |
| **Embed Links** | So Waver can send rich embeds with track info. |

## Step 2: Join a Voice Channel

Open Discord and join any voice channel in the server where you just added Waver.

## Step 3: Play Your First Song

In any text channel that Waver can read, type the following:

```bash
p!play lofi hip hop radio
```

Waver will automatically:
1. Join your voice channel.
2. Search for the song.
3. Start playing it immediately.

## Step 4: Control Playback

Now that music is playing, here are the essential commands:

| Command | What it does |
|---|---|
| `p!pause` | Pauses the current track. |
| `p!resume` | Resumes playback. |
| `p!skip` | Skips to the next song in the queue. |
| `p!stop` | Stops playback and disconnects the bot. |
| `p!queue` | Shows the current queue. |
